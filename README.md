这是一个完全开源免费的Tally灯项目

除了传统的LED指示方式，还提供了手机iPad等设备的WebTally方式

可通过网页进行设置，可以在线升级，目前使用WiFi方案，后续考虑不改变架构的情况下增加ESP-Now、RF(公里级传输)支持

目前支持时代奥视OSEE、vMix切换台，芯象即将支持，BMD ATEM系列切换台已支持但是因为缺少测试机测试会晚些发布，欢迎各厂商联系支持对接事宜(互免)
